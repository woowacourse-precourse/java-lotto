# 🚀 숫자 야구 게임 구현에 필요한 기능의 목록

## 🎯 중점을 두어야 하는 부분
3주 차 미션 메일에서 언급한 내용을 바탕으로 기능을 구현하기 위해 노력했습니다.
1. 클래스(객체)를 분리하는 연습
2. 도메인 로직에 대한 단위 테스트를 작성하는 연습

### 🔍 클래스(객체)를 분리하는 연습
#### 기능 구현을 위해 필요한 메소드
    사용자로부터 로또 구입 금액을 입력 받는 메소드
    쉼표를 기준으로 구분된 당첨 번호를 입력 받는 메소드
    보너스 번호를 입력 받는 메소드
    오름차순 정렬을 위한 메소드
    발행한 로또 수량 및 번호를 출력하는 메소드
    당첨 내역을 출력하는 메소드
