# 기능 목록

## 입력
+ [✅] 구입 금액 입력
+ [✅] 당첨 번호 입력
+ [] 보너스 번호 입력
+ [] 구입 금액이 1000원으로 나누어 떨어지지 않는 경우 예외 처리
+ [] 입력한 번호가 1~45 범위 밖일 경우 예외 처리

## 출력
+ [] 발행된 로또 번호 출력
+ [] 당첨 내역 출력
+ [] 수익률 출력
+ [] 예외 상황 시 에러 메시지 출력

## 로또 게임
+ [] 구입 금액만큼 로또 발행
+ [] 중복되지 않는 1~45 범위 6개의 숫자를 생성하여 로또 발행
+ [] 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 계산
+ [] 수익률 계산
