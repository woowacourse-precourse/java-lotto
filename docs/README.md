##구현할 기능 목록
Lotto
- [X] 로또 1개 생성
  - [ ] 1~45 사이 숫자인지 검사
  - [X] 전체가 6개 숫자인지 검사
  - [X] 중복된 숫자가 있는지 검사

Person
- [X] 로또 구매
  - [ ] 1000원 단위인지 검사 
  - [X] 랜덤 6개 숫자로 로또 생성
    - [X] 번호 생성시 오름차순 정렬
- [X] 총 당첨 금액 계산
- [X] 수익률 계산


Game
- [X] 당첨번호 및 보너스 번호 생성
- [X] 등수 계산
- [X] 등수에 따른 당첨 금액 반환
- [X] 당첨 내역 저장
