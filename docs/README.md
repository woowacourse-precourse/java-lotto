### 예외 처리

- [ ] 로또 번호

  - [ ] 로또 번호는 1부터 45 사이여야 한다.
    - "[ERROR] 로또 번호는 1부터 45 사이의 자연수여야 합니다."
  - [ ] 로또 번호는 6개여야 한다.
    - "[ERROR] 로또 번호는 6개여야 합니다."

  - [ ] 로또 번호에 중복이 없어야 한다.

  - [x] 구입금액은 1000원으로 나누어 떨어져야 한다. - `validatePurchaseMoney`
