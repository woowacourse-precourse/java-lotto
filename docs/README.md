## 구현 기능 목록
### 로또 번호 자동 생성기
- [x] 구매자가 구매한 금액 만큼 로또를 자동으로 발급한다.
  - [x] 1 ~ 45 중 중복되지 않도록 6개 숫자 뽑기
  - [x] 뽑은 번호를 오름차순으로 정렬한다.
  - [x] 뽑은 로또를 장수 만큼 리턴한다.
### 로또 판매자
- [X] 구매 금액만큼 로또 발급 : 발급 개수 반환
### 로또 구매자
- [x] 구매 금액 지불
  - 예외 : 1,000 원 단위로 지불하지 않은 경우
  - 예외 : 숫자가 아닐 경우 예외
  - String 을 int 타입으로 변환하는 기능
### 로또 번호 지정자
- [x] 6자리 당첨 번호 입력받기
  - 예외 : 숫자가 아닐 경우
  - 예외 : 6 자리가 아닐 경우
  - 예외 : 1 ~ 45 범위를 벗어날 경우
  - 예외 : 중복 숫자 있을 경우
- [x] 보너스 번호 입력받기
  - 예외 : 숫자가 아닐 경우
  - 예외 : 1 ~ 45 범위를 벗어날 경우
  - 예외 : 로또 번호와 중복 숫자 있을 경우
### 로또 결과 판정자
- [x] 구매한 로또 번호와 정답 로또 비교하여 등수 판단
  - [x] 판단한 결과에 따라 Enum LottoResult 에 반영
### 당첨 결과 Enum
- [x] 1등, 2등, 3등, 4등, 5등 의 메시지와 당첨금액 보유
  - [x] 최종 당첨 금액 반환 기능
  - [x] 등수마다 최종 당첨 장수 반환 기능
### 수익률 계산기
- [x] 수익율 구해서 반환
### 로또 스토어
- [x] 로또 판매,구매 로직이 작동하는 곳
