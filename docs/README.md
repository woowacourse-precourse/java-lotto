# 기능 목록

## 게임 컨트롤러
1. 웰컴 메세지를 출력한다.
2. 구입 금액을 입력받는다.
3. 로또 서비스에 입력을 넘겨준다.
4. 로또 서비스에서 던져주는 익셈션을 핸들링한다.
5. 로또 리스트에 있는 로또 번호를 출력한다.

## 에러 메세지
1. enum 클래스로 제작한다.
2. 각종 에러 메세지를 저장하고 있다.

## 일반 메세지
1. enum 클래스로 제작한다.
2. 일반적인 출력 메세지를 저장한다.

## 로또 서비스
1. 올바른 입력인지 체크하고 아니라면 익셉션을 리턴한다. 숫자가 아닌것, 천원단위가 아닌것
2. 입력을 money로 환산한다.
3. 돈으로 전부 로또를 산다.

## 로또
1. get메소드를 통해 로또 정보를 요청한다.
