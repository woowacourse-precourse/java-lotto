# 🚀 미션 - 로또

## ✍ 기능 구현 목록

### ✅ 입력
- [x] 구입 금액 입력
- [x] 당첨 번호 입력
- [x] 보너스 번호 입력

### ✅ 출력
- [x] 구입 안내 출력
- [x] 구입 결과 출력
- [x] 구입한 로또 번호 출력
- [x] 당첨 번호 입력 안내 출력
- [x] 보너스 번호 입력 안내 출력
- [ ] 당첨 통계 출력
- [x] 수익률 출력
- [ ] 에러 출력

### ✅ 로또
- [x] 로또 발행
  - [x] 구입 금액에 따른 로또 개수 확인
    - [x] 구입 금액이 1000원 단위가 아니라면 예외 발생
    - [x] 중복된 번호 존재 시 예외 발생
    - [x] 범위 밖의 번호 존재 시 예외 발생
- [ ] 당첨 번호 추첨
- [ ] 보너스 번호 추첨

### ✅ 계산
- [ ] 당첨 확인
- [ ] 수익률 계산
