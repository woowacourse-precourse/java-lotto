- 로또 번호 유효성 검사
    - 개수가 맞는지, 숫자인지, 중복된 건 없는 지, 1~45까지 인지

- 정답 번호 입력받기
    - 로또 번호 유효성 검사하기

- 보너스 번호 입력 받기
    - 숫자인지, 1~45까지인지 검사하기

- 돈 입력 받기
    - 1000원 단위인지, 숫자인지 확인하기

- 랜덤 로또 번호 생성하기
    - 1~45까지 랜덤으로 유니크한 6개의 숫자 뽑기

- 돈 넣으면 그만큼 로또 번호 생성해주기
    - 1000원 개수만큼 랜덤 로또 번호 생성하기

- 랭킹 계산하기
    - 뽑은 숫자들이 정답과 몇개가 겹치는 지 계산하기
    - 뽑은 숫자중에 보너스 넘버가 있는 지 확인하기
    - 겹치는 개수를 토대로 적절한 랭킹 리턴해주기

- 당첨 내역 계산하기
    - 각 등수에 대한 티켓 개수 세기

- 수익률 계산하기
    - (총 당첨금액 - 매수금액) / 매수금액
