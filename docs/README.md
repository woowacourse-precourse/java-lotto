# 미션 - 로또

## 기능 목록

### 1.로또 구입 금액

+ 로또 가격은 1000원
+ (e) 1000원 단위 외의 금액 불가
+ (e) 문자열 입력
+ 구매 로또 개수 구하기

### 2.로또 목록

+ 로또 생성
    + 숫자는 1 ~ 45
    + (e) 중복된 숫자 불가
    + Lotto 클래스 활용
+ 목록 출력
+ 로또 숫자 정렬

### 3.구매한 로또 번호들 출력

### 4.당첨 번호 입력

+ 6개 숫자 입력 (ex: 1, 2, 3)
+ (e) 숫자는 1 ~ 45
+ 입력 구분자는 ,
+ (e) 중복된 숫자 불가

### 5.보너스 번호 입력

+ 숫자는 1 ~ 45
+ (e) 당첨 번호와 중복 불가

### 6.비교

+ 숫자랑 위치 일치하는지 확인
+ 보너스볼 확인

### 7.비교 결과 출력

+ 수익률 출력 (수익률 = 당첨 금액 /구입 금액)
