# 기능 목록
## 기능 요구 사항
| No. | 기능                                                   |
|-----|------------------------------------------------------|
| 1.  | 구입 금액 입력<br/>-1000 단위 숫자                             |
| 2.  | 랜덤 로또 번호 생성                                          |
| 3.  | 당첨 번호 입력<br/>-숫자 6개(쉼표로 구분)<br/>-중복 X<br/>-범위 1 ~ 45 |
| 4.  | 보너스 번호 입력<br/>-범위 1~45<br/>-중복 X                     |
| 5.  | 출력 : 구매한 로또 번호 리스트, 당첨내역, 수익률                        |

|-----|-------------------------------------------------------------------------|
