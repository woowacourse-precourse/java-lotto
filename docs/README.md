# 3주차 기능 목록

### 입력
```
1. 사용자 입력

-사용자가 로또 금액을 입력한다. 구입 금액이 1000원 단위가 아닐 경우 IllegalArgumentException를 발생시켜

2. 게임기 입력

-게임기는 당첨 번호를 입력한다. 번호는 쉼표를 기준으로 구분한다.
-예외상황) 1. 숫자 범위가 1~45사이가 아닐 경우 에외처리를 한다.
          2. 문자가 있을 경우
          3. 6개 입력을 받지 않았을 경구
-마지막으로 보너스 번호를 입력받는다.
-예외상황) 1. 숫자 범위가 1~45사이가 아닐 경우 에외처리를 한다.
          2. 문자가 있을 경우
          3. 1개 입력을 받지 않았을 경구
```

### 출력
```
-구매한 로또 수량 만큼 번호를 출력하는데 오름차순으로 정렬하여서 보여준다.

-맞춘 번호 갯수 별로 당첨 내역을 출력한다.

-맞춘 갯수를 기반으로 총 수익률을 소수점 첫째 자리까지 출력한다.(둘째 자리에서 반올림한다.)
```

### 예외 처리
```
- 예외 상황 발생 시 [ERROR]"로 시작하는 에러 문구를 출력한다. 
```
