#로또

### 기능 목록

- [ ] 로또를 구입 금액에 따라 발행한다.
  - [ ] 로또 발행 개수를 구한다.
  - [ ] 1부터 45까지의 서로 다른 임의의 숫자 6개를 생성한다.
  - [ ] 오름차순으로 정렬한다
- [ ] 당첨 번호와 발행 번호를 비교한다.
  - [ ] 몇개의 번호가 같은지 구한다.
  - [ ] 보너스 번호와 같은지 비교한다.
- [ ] 로또의 등수를 구한다.
- [ ] 수익률을 구한다.
  - [ ] 총 수익을 구한다.

#### 에러 상황
- [ ] 로또 구입 금액입력
  - [ ] 숫자가 아닌 경우
  - [ ] 1000원 단위가 아닌 경우
- [ ] 당첨번호 입력
  - [ ] 숫자와 쉼표말고 다른 문자를 입력받은 경우
  - [ ] 입력 포맷이 "숫자,숫자,숫자..."이 아닌 경우
  - [ ] 입력 길이가 11이 아닌 경우
  - [ ] 숫자가 중복되는 경우
- [ ] 보너스 번호 입력
  - [ ] 숫자가 아닌 경우
  - [ ] 한글자가 아닌 경우
  - [ ] 당첨번호와 중복되는 경우
