# 로또 게임
본 프로젝트는 우아한 테크코스 웹 백엔드 과정의 프리코스 3주차 과제이다.
본 문서는 해당 과제의 기능 목록을 정리한 문서이다.

# 기능 목록
- [x] 구입 금액 받기
- [x] 당첨 번호 받기
  - [x] 쉼표로 나눠야 한다
- [x] 보너스 번호 받기
- [x] 숫자인지 검사
  - [x] 모두 양수이다
  - [x] 첫번째 숫자는 0일 수 없다
- [x] 구입 금액 검증
  - [x] 로또 가격으로 나누어 떨어져야 한다
- [x] 로또 구입
- [x] 로또 발행
- [x] 로또 번호 검증
  - [x] 크기가 6이여야 한다
  - [x] 범위 내에 있어야 한다
  - [x] 중복이 없어야 한다
- [x] 로또 번호 출력
- [] 보너스 번호 검증
  - [] 범위 내에 있어야 한다
  - [] 당첨 번호와 중복이 없어야 한다
- [] 로또 번호 비교
- [] 수익률 계산
- [] 결과 추가
  - [] 5개가 일치할 경우 보너스 일치 여부 확인해야 한다
