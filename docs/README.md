## ✏️ 구현할 기능 목록

1. 로또 구입 금액 입력 받기
   1. 1000원으로 나누어 떨어지지 않는 경우 예외 처리
2. 구입 금액에 따른 발행해야 하는 로또 수량 계산
3. 당첨 번호 생성 (중복되지 않는 6개 숫자 + 보너스 번호 1개)
4. 로또 발행 (중복되지 않는 6개 숫자 뽑음)
5. 당첨 번호 입력 받기
   1. 1~45 아니면 예외 처리
6. 보너스 번호 입력 받기
   1. 1~45 아니면 예외 처리
   2. 당첨 번호와 중복이면 예외 처리
7. 당첨 여부 확인
   1. 사용자가 구매한 로또 번호와 당첨 번호 비교
8. 수익률 계산 (소수점 둘째 자리에서 반올림)
9. 당첨 내역 출력
10. 수익률 출력
11. 예외 상황시 에러 문구 출력
