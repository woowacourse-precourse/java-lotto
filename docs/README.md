1. 로또 구입 금액 입력받기
   - 1,000원 단위인지 확인 
   - 구입 금액 > 0원 인지 확인
2. 입력받은 금액만큼 로또 발행 
   - 로또 번호가 6개의 숫자로 이루어져 있는지 확인
   - 로또 번호에 중복된 숫자가 있는지 확인
   - 1부터 45 사이의 수로 구성되었는지 확인
3. 발행한 로또 출력
4. 당첨 번호 입력받기
   - ,를 기준으로 구분 시 6개의 숫자가 제대로 입력되었는지 확인
   - 중복되지 않은 숫자인지 확인
   - 1부터 45 사이의 수인지 확인
5. 보너스 번호 입력받기
6. 발행한 로또 수량 및 번호 출력
   - 로또 번호 오름차순 정렬하여 출력
7. 당첨 내역 및 수익률 계산
   - 당첨 내역 출력
   - 수익률은 소수점 둘째 자리에서 반올림
