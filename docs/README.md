## 입력 처리

- [x] 사용자 구입 금액 입력 처리 및 예외 처리
- [x] 당첨번호와 보너스 번호 입력 처리 및 예외 처리

## 출력 처리

- [x] 사용자 구매 갯수 출력
- [X] 사용자가 구매한 로또의 번호 출력
- [x] 사용자의 당첨 갯수를 출력
- [x] 사용자의 이익률을 출력

## 메인 로직
- [x] 컨트롤러 생성 및 입력 로직 추가
- [X] 서비스 레이어 생성 및 사용자 로또 구매 갯수 처리
- [X] 컨트롤러에 서비스 의존성을 주입
- [X] 사용자가 구매한 로또 생성
- [x] 보너스 번호 예외처리 및 보너스 번호를 포함한 로또
- [x] 맞은 로또를 등수에 따라 분류
- [X] 맞은 금액에 따른 이익률 계산
- [x] 메인 클래스에서 GameController 실행
