# 미션 - 로또
## 미션 분석
1. 사용자에게 로또 구입 금액을 입력받는다.
2. 로또 구입 금액에 맞춰 로또를 발행한다.
3. 로또 당첨 번호와 보너스 번호를 입력받는다.
4. 사용자 로또 번호와 로또 당첨 번호를 비교한다.
5. 결과에 맞춰서 등수와 수익률을 출력한다.
> 수익률(%) = 당첨 금액/로또 구입 금액 * 100
