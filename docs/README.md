### 입력 
- [ ]  로또 구입 금액을 입력 받기.
> - [ ]  `camp.nextstep.edu.missionutils.Console`의 `readLine()`을 활용하기.
> - [ ]  숫자가 아닌 값을 입력 할 경우 예외 처리.
> - [ ]  1,000원으로 나누어 떨어지지 않을 경우 예외 처리.
> - [ ]  1,000원 미만의 값을 입력할 경우 예외 처리.

- [ ]  당첨 번호를 입력 받기
> - [ ]  쉼표로 구분되지 는 경우 예외 처리.
> - [ ]  쉼표와 숫자가 아닌 값일 경우 예외 처리.
> - [ ]  6개의 숫자가 아닌 경우 예외 처리.
> - [ ]  45 초과, 1미만의 범위의 수를 입력한 경우 에외 처리.
