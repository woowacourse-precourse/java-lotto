# 미션 - 로또

## 기능 목록
[x] 로또 구입 금액 입력 받기
  - (예외상황) 1,000원으로 나누어 떨어지지 않는 경우 에러 메세지 출력 후 종료
[x] 로또 발행 번호 뽑기
[x] (사용자) 당첨 번호 입력 받기
  - (예외상횡) 숫자 번호 범위(1~45) 넘어가면 예외 처리
  - (예외상황) 중복하는 번호 입력시 예외처리
[x] (사용자) 보너스 번호 입력 받기
  - (예외상황) 숫자 번호 범위(1~45) 넘어가면 예외 처리
  - (예외상황) 중복하는 번호 입력시 예외처리
[x] 발행한 로또 수량 및 번호 출력하기 
[x] 당첨 내역 출력하기 
[x] 수익률 출력하기
