> ## 💰 3주차 로또 게임

---
## 기능 요구 사항
-[x] 로또 구입 금액을 입력한다
  - 천원 단위로 입력받으며, 1000으로 나누어 떨어지지 않으면 예외처리한다
-[x] 구입 금액 만큼 로또를 발행한다
  - 1 ~ 45 까지의 숫자를 랜덤으로 6개 선별한다
-[x] 로또 게임의 갯수만큼 당첨 번호를 저장한다
-[x] 당첨 숫자 6개와 보너스 번호 1개를 뽑는다
-[x] 각 로또 게임 마다 당첨 번호를 확인한다
-[x] 당첨 내역을 확인해 등수별로 카운트한다
-[x] 당첨 내역을 토대로 수익률을 계산한다
