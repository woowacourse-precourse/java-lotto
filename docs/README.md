로또 구매 관련
---
- 로또 구매 금액 입력
- 로또 번호 생성
- 당첨 번호 입력
- 보너스 번호 입력
- 당첨 통계 출력
- 수익률 계산 후 출력

예외 사항
---
  - 구매 금액 관련
    - 로또 구매 금액이 1000의 배수인지 확인
  - 당첨 번호 관련
    - 입력 받은 당첨번호가 총 6개의 숫자가 아니면 예외 처리
    - 입력 받은 당첨번호가 모두 1~45 사이가 아니면 예외 처리
    - 입력 받은 숫자 중에 중복이 있을 경우 예외 처리
  - 보너스 번호 관련
    - 숫자가 아닐 경우 예외 처리
    - 1~45 사이의 숫자가 아니면 예외 처리
    - 입력 받은 번호와 겹칠 경우 예외 처리
