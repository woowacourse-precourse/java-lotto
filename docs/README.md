# 👾로또 게임👾

---

### 구현해야 할 컴포넌트 목록

- #### 로또 머신
- #### 로또
- #### 당첨 번호

---

### 컴포넌트 별 메서드 목록

- **로또 머신**
    * [x] 로또 구매하기(금액 입력받기 및 로또 배열 초기화)
    * [x] 구매한 로또들 출력하기
    * [x] 당첨 번호 초기화하기(당첨 번호 입력받기)
    * [x] 구매한 로또들 결과 기록하기
    * [x] 입력 금액 유효성 검증하기
    * [x] 결과 및 수익률 출력하기

  
- **로또**
    * [x] 로또 생성 시 입력값 유효성 검증하기
    * [x] 정렬된 로또 번호 출력하기


- **당첨 번호**
    * [x] 당첨 번호 생성 시 입력값 유효성 검증하기
    * [x] 당첨 번호와 로또를 비교하여 등급 판별하기
