# 🔍 기능 목록 정의 🔍

## ****1. LottoManager****

- [ ]  전체 로직 관리
- [ ]  로또 결과 산출
- [ ]  수익률 구하기

## 2. Input

- [ ]  구입 금액 입력
- [ ]  당첨 번호 입력
- [ ]  보너스 번호 입력

## 3. Lotto

- [ ]  숫자 비교
- [ ]  번호 길이 검증
- [ ]  숫자 범위 검증
- [ ]  공백 검증
- [ ]  중복 검증
- [ ]  번호 타입 검증

## 4. LottoWinningNumber

- [ ]  숫자 중복 확인
- [ ]  포함관계 활용하여 로또에서 검증

## 5. Money

- [ ]  공백 검증
- [ ]  타입 검증
- [ ]  1000원 미만 검증
- [ ]  숫자 나누기

## 6. LottoFactory

- [ ]  금액 만큼 6자리 숫자 생성
- [ ]  오름차순으로 정렬

## 7. LottoNumbers

- [ ]  로또 객체로 관리

## 8. BonusNumber

- [ ]  String 으로 입력 받은 보너스 번호 int 로 변환 검증
- [ ]  보너스 번호 길이 검증
- [ ]  보너스 숫자 타입 검증

## 9. **Ranking**

- [ ]  연관됨 상수 enum 처리
- [ ]  놓은 값 찾기

## 10. LottoResult

- [ ]  상금 확인

## 11. Output

- [ ]  로또 구매 개수 출력
- [ ]  로또 구매 개수 비례 하여 번호들 출력
- [ ]  당첨 통계 출력
- [ ]  당첨 결과 출력
- [ ]  수익률 출력
