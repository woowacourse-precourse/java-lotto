## 기능 목록

1. 적절한 메시지를 출력하고 사용자의 입력을 받는 기능(UI)


2. 로또 구입 금액에 대한 유효성 검사 기능


3. 구입 금액에 해당하는 만큼 로또를 발행하는 기능


4. 발행한 로또들을 번호 순서대로 정렬하는 기능


5. 정렬한 로또 복권들을 출력하는 기능(UI)


6. 당첨 번호에 대한 유효성 검사 기능


7. 보너스 번호에 대한 유효성 검사 기능


8. 당첨 여부를 체크하는 기능


9. 당첨 내역을 출력하는 기능(UI)


10. 수익률을 계산하는 기능


11. 수익률을 출력하는 기능(UI)
