# 미션 - 숫자 야구 게임

## 기능 구현 목록
- [X] 구입금액 입력 받기
- [X] 몇개의 로또를 샀는지 계산
- [X] 입력한 금액만큼 로또번호 출력하기 + 로또번호 정렬하기
- [X] 당첨번호 입력받기
- [X] 보너스 번호 입력받기
- [X] 당첨통계 출력하기
- [X] 총 수익률 계산하기

- 예외처리
   -[X] [예외처리] 입력금액 1000원단위 아닐시 예외처리
   -[X] [예외처리] 당첨번호 및갯수 보너스 번호가 범위 외일 경우 예외처리
   -[X] [예외처리] 당첨번호 및 보너스 번호가 중복될 시 예외처리
   - [X] [예외처리] 당첨번호가 6개가 아닐때 예외처리
   - [X] [예외처리] 입력된 값이 숫자가 아닐 시 예외처리
