## 전반적인 개요

1. 로또
    1. 로또 번호 생성 클래스
        - 무작위 번호 6개를 오름차순으로 List\<integer> 로 생성
            1. 번호가 6개이다.
            2. 번호가 1~45 범위 안이다.
            3. 번호의 중복이 없다.
            4. 번호들이 등장하는 비율이 모두 1/45 이다.
            5. 번호가 오름차순이다.
    2. 입력 받은 당첨 로또 번호 클래스 (**Lotto**)
        - 사용자에게 입력받은 로또 번호를 에러를 관리
            1. 숫자가 6개가 아닐 때 에러 발생, 에러 문구 출력
            2. 숫자가 1~45 범위가 아닐때 에러 발생, 에러 문구 출력
            3. 숫자에 중복이 있을 때 에러 발생, 에러 문구 출력
            4. 생성자에서 입력받은 List의 값이 변하지 않고 저장되고 저장된 List를 반환해주는 기능
2. 돈
    1. 입력 받은 구입 금액 클래스
        - 입력 받은 금액의 에러와 로또 구매 수 관리
            1. 숫자가 로또 한 장 가격의 배수가 아닐 경우 에러 발생
            2. 로또 구매 갯수 반환 기능
            3. 금액 반환 기능
3. 당첨 통계 및 수익률
    1. 로또 번호 일치율 관리 클래스
        - 당첨 여부 확인 후 당첨 갯수 저장
            1. bonus 번호 입력받아 에러 확인하기
                - 1~45 범위의 숫자
                - Lotto와 중복 없음
            2. 입력받은 List\<List\<Integer>>와 Lotto를 활용하여 6개 일치하는 1등 확인
            3. 5개만 일치하고 bonus 번호를 가지고 있는 경우인 확인 2등 확인
            4. 5개만 일치하고 bonus 번호는 없는 3등 확인
            5. 4개만 일치하는 경우인 4등 확인
            6. 3개만 일치하는 경우인 5등 확인
            7. 각 등수를 key로 등수의 당첨 수를 value로 Map으로 저장하여 리턴
    2. 등수별 갯수(Map)를 입력 받아 당첨 내역과 수익률 출력
        - 입력 받은 Map을 활용하여 수익률을 계산하는 클래스
            1. 등수별 금액을 합산하여 수익률을 계산하여 리턴
        - 입력 받은 Map을 활용하여 당첨 내역과 수익률을 출력
            1. 5개 일치, 보너스 볼 일치(2등) 출력
            2. 3~6개 일치(5,4,3,1등) 출력
            3. 수익률 출력
4. 입력 관리
    1. 변환 클래스
        - 입력받은 String 번호를 List\<integer>형으로 변환해 리턴
            1. 잘못된 입력이 주어진 경우 에러 발생
                - 숫자가 아닌 경우
                - 입력이 없는 경우
            2. , 단위로 쪼개서 List\<integer> 리턴
        - 입력받은 String 금액을 int형으로 변환해 리턴
            1. 잘못된 입력이 주어진 경우 에러 발생
                - 숫자가 아닌 경우
                - 입력이 없는 경우
5. 어플리케이션
    - 전반적인 흐름 관리

#### Lotto 클래스

입력받은 로또 번호를 관리하는 클래스이다.

- 필드

| 이름                       | 내용                    |
|--------------------------|-----------------------|
| List\<Integer> *numbers* | 입력받은 번호를 담아두기 위한 것이다. |

- 메서드

| 이름                                       | 내용                                      |
|------------------------------------------|-----------------------------------------|
| **Lotto**(List\<Integer>)                | 예외를 검사하고 입력받은 리스트를 *number*에 복사해 담아둔다.  |
| **validate**(List\<Integer>)             | 예외를 검사하는 메서드로 길이, 중복, 범위를 체크한다.         |
| boolean **isWrongSize**(List\<Integer>)  | 길이가 6인지 판별하여 6이 아니면 true를 반환한다.         |
| boolean **isDuplicate**(List\<Integer>)  | 중복이 있으면 true를 반환한다.                     |
| boolean **isOutOfBound**(List\<Integer>) | 1~45 범위에서 벗어나는 숫자가 하나라도 있으면 true를 반환한다. |
| List\<Integer> **getNumbers**()          | *numbers*를 복사하여 반환한다.                   |

### result

#### **LottoCompare** 클래스

입력받은 로또 번호와 bonus 번호를 보고 예외를 체크하고 QuickPicks를 비교한 결과를 Map으로 만들어서 리턴한다.

- 필드

| 이름            | 내용                |
|---------------|-------------------|
| int[] *count* | 등수별 당첨수를 담는 배열이다. |

- 메서드

| 이름                                                         | 내용                                                      |
|------------------------------------------------------------|---------------------------------------------------------|
| **LottoCompare**(List\<List<Integer>>,List\<Integer>, int) | 예외를 체크하고 당첨 횟수를 *count*에 저장한다.                          |
| **validate**(List\<Integer>, int)                          | bonus 번호의 예외를 확인한다. 중복과 범위를 체크하여 예외를 발생시킨다.             |
| **checkingNumber**(List<Integer>,List\<Integer>, int)      | 번호를 비교하여 등수를 알아내 *count*에 더해준다.                         |
| Map\<String, Integer> **getPrizesMap**()                   | 등수 이름을 key로 *count*에 저장된 값(당첨수)을 value로 Map을 만들어서 반환한다. |

--------------------------------

#### **LottoPrizes** enum

로또의 등수명, count의 인덱스, 일치율, 상금을 담고있는 열거형 클래스이다. 일치율과 보너스 번호 유무로 등수를 판단할 수 있다.

- 필드

| 이름        | 내용                                          |
|-----------|---------------------------------------------|
| *FIRST*   | 1등이며 6개가 일치하고 상금은 2000000000이고 인덱스 번호는 1이다. |
| *SECOND*  | 2등이며 5개가 일치하고 상금은 30000000이고 인덱스 번호는 2이다.   |
| *THIRD*   | 3등이며 5개가 일치하고 상금은 1500000이고 인덱스 번호는 3이다.    |
| *FOURTH*  | 4등이며 4개가 일치하고 상금은 50000이고 인덱스 번호는 4이다.      |
| *FIFTH*   | 5등이며 3개가 일치하고 상금은 5000이고 인덱스 번호는 5이다.       |
| *NOTHING* | 당첨이 없는 경우를 의미한다.                            |

- 메서드

| 이름                                  | 내용                                |
|-------------------------------------|-----------------------------------|
| LottoPrizes **prizes**(int,boolean) | 일치율과 보너스 번호 포함 여부로 등수를 판별에 반환해준다. |
| LottoPrizes **prizes**(int)         | 일치율로 2등을 제외한 등수를 판단해 반환한다.        |

#### LottoYield 클래스

로또의 이익률을 계산하는 클래스로 입력받은 Map을 바탕으로 이익률을 계산한다.

- 필드

| 이름              | 내용             |
|-----------------|----------------|
| *totalWinnings* | 총 상금을 저장한다.    |
| *totalPurchase* | 총 구매 비용을 저장한다. |

- 메서드

| 이름                                        | 내용                                                                                                         |
|-------------------------------------------|------------------------------------------------------------------------------------------------------------|
| **LottoResult**(Map<String, Integer>)     | *totalWinnings*와 *totalPurchase*를 초기화하고 plusToTotalWinnings()와 plusToTotalPurchase()를 호출해 구매금액과 총상금을 저장한다. |
| **plusToTotalWinnings**(LottoPrizes, int) | 등수의 상금과 당첨 개수를 곱해 *totalWinnings*에 더해준다.                                                                   |
| **plusToTotalPurchase**(int)              | 구매금액과 개수를 곱해 *totalPurchase*에 더해준다.                                                                        |
| double **yield**()                        | 수익률을 반환한다.                                                                                                 |

#### InputConvert 클래스
입력으로 받은 String의 예외를 확인하고 필요한 형태에 맞게 변환시켜주는 클래스
- 메서드

| 이름                                               | 내용                                                  |
|--------------------------------------------------|-----------------------------------------------------|
| List\<Integer> **lottoNumber**(String)           | 입력받은 로또 번호의 예외를 확인하고 List\<Integer>로 변환하여 반환한다.     |
| **lottoValidate**(String)                        | 입력의 길이가 0인지, ,나 숫자 이외의 입력은 없는지 확인하여 예외를 발생시킨다.      |

| **oneNumberValidate**(String)                    | 숫자 하나의 입력이 길이가 0인지 숫자 이외의 입력은 없는지 확인하고 예외를 발생시킨다.   |
