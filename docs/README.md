# 👻 기능목록 

---

- 로또 랜덤숫자 생성
  - 1 ~ 45 범위의 랜덤숫자 생성한다.


- 입력 / 출력
    - 입력
        - 금액을 입력 받는다.
        - 당첨 번호를 입력 받는다.
        - 보너스 번호를 입력 받는다.
    - 출력
        - 금액만큼 구입 한 로또 번호를 출력한다.
        - 당첨 내역을 출력한다.
        - 수익률을 출력한다.


- 검증 / 예외
    - 구입 금액 검증
    - 로또 번호 검증
    - 당첨 번호 검증
    - 보너스 번호 검증

  위 검증을 통해서 사용자가 잘못된 값을 입력할 경우`IllegalArgumentException`를 발생시키고, "[ERROR]"로 시작하는 에러 메시지를 출력 후 종료한다.


- 비교
    - 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 및 수익률을 계산한다.
