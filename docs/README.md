### [ 진행 ]

- [x] 로또 발행 진행
- [x] 당첨 여부 판별 진행

### [ 로또 ]

- [x] 당첨번호 발행
- [x] 로또번호 발행
    - 로또 숫자 생성
        * 조건
            * 숫자범위 지키기
            * 중복하지 않기
            * 6개 뽑기

- [x] 로또 종이 발행
    * 조건
        * 1장 가격 1000원 단위
        * 돈 받은 만큼만 발행
        * 1000으로 나눠떨어져야함함

#### 로또 당첨 판별

- [x] 당첨여부 계산
- [x] 수익률 계산
