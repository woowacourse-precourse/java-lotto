## Application Class

### 로또 구입 금액 입력 기능

|        | 설명                                                      |
|--------|:--------------------------------------------------------|
| 기능     | "구입금액을 입력해 주세요." 라는 메시지를 출력후 사용자가 로또를 구입할 금액을 입력을 받습니다. |
| 메소드 이름 | receiveBuyPrice                                         |
| 매개변수   | -                                                       |
| 반환 결과  | 로또 구입 금액                                                |
| 예외 사항  | 1000원으로 나누어 떨어지지 않는 경우 `IllegalArgumentException` 발생    |

### 당첨번호 입력 기능

|        | 설명                                                                    |
|--------|:----------------------------------------------------------------------|
| 기능     | 당첨 번호를 ,로 구분해 6개의 숫자를 입력합니다.                                          |
| 메소드 이름 | receiveWinningNumbers                                                 |
| 매개변수   | -                                                                     |
| 반환 결과  | 로또 번호 1~45 숫자 6자리를 가지는 Integer 리스트                                    |
| 예외 사항  | ,로 숫자를 구분하지 않거나 1~45의 숫자가 아닌 값을 입력하는 경우 `IllegalArgumentException` 발생 |

### 당첨 번호를 파싱하는 기능

|        | 설명                                                                    |
|--------|:----------------------------------------------------------------------|
| 기능     | 사용자로부터 입력받은 당첨번호를 파싱해 Integer List로 반환합니다.                            |
| 메소드 이름 | parseWinningNumbers                                                   |
| 매개변수   | 사용자 입력                                                                |
| 반환 결과  | 로또 번호 1~45 숫자 6자리를 가지는 Integer 리스트                                    |
| 예외 사항  | ,로 숫자를 구분하지 않거나 1~45의 숫자가 아닌 값을 입력하는 경우 `IllegalArgumentException` 발생 |

### 보너스 번호 입력 기능

|        | 설명                                                    |
|--------|:------------------------------------------------------|
| 기능     | 보너스 번호 1개의 숫자를 입력받습니다.                                |
| 메소드 이름 | receiveBonusNumber                                    |
| 매개변수   | -                                                     |
| 반환 결과  | 보너스 번호                                                |
| 예외 사항  | 1~45의 숫자가 아닌 값을 입력하는 경우 `IllegalArgumentException` 발생 |

### 당첨 결과를 계산하는 기능

|        | 설명                                    |
|--------|:--------------------------------------|
| 기능     | 로또 결과별 횟수를 계산해 Map으로 반환합니다.           |
| 메소드 이름 | compareLotteriesResult                |
| 매개변수   | 로또 리스트, 당첨 번호, 보너스 번호                 |
| 반환 결과  | 로또 결과를 Key로 가지고 당첨 횟수를 Value로 가지는 Map |
| 예외 사항  | -                                     |

### 당첨 결과 출력 기능

|        | 설명                                                                   |
|--------|:---------------------------------------------------------------------|
| 기능     | 당첨 결과를 `n개 일치 (당첨 금액) - x개` 형식대로 출력합니다. <br/> 그리고 마지막에 총 수익률을 출력합니다. |
| 메소드 이름 | printResult                                                          |
| 매개변수   | 당첨 결과 List, 수익률                                                      |
| 반환 결과  | -                                                                    |
| 예외 사항  | -                                                                    |
