## 🚀 기능 요구 사항

1. 구입 금액 입력
2. 구입 금액 만큼 로또 발행
   - 로또 번호 출력 오름차순 정렬
3. 당첨 번호 + 보너스 입력
4. 사용자 로또랑 당첨 번호 비교
5. 수익률 계산 및 출력
   - 당첨 내역 출력
   - 수익률 소수점 둘째 자리에서 반올림
6. 예외 처리(에러 문구 출력)
    - 숫자 범위 1~45
   - 구입 금액 1000 단위로 입력
   - 당첨 번호 (1,2,3,4,5,6) 형식으로 입력
   - 보너스 번호 1~45 당첨 번호와 겹치지 않는 한개의 수
