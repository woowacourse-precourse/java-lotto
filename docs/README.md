## 📝 구현 기능 목록

- [x] **`로또 시작 문구`를 출력하는 기능** - InputView

```
구입금액을 입력해 주세요.
```

- [x] **금액을 입력받는 기능 (Console API)**
    - [x] 문자열로 입력 받은 수를 int형으로 변환하는 기능
> 예외

- [x] 1,000단위로 나누어 떨어지지 않을 경우
- [x] 1,000원보다 작을 경우
- [x] 숫자가 아닐 경우

> 처리 (순서대로)

1. `IllegalArgumentException` 발생
2. `[ERROR]`로 시작하는 에러 메시지를 출력
3. 애플리케이션 종료

- [x] **`로또 구입 개수 문구`를 출력하는 기능** - OutputView

