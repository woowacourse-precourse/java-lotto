# 기능 목록
1. input
   - 로또 구입 금액
   - 당첨번호
   - 보너스 번호
2. Random number 생성
   - 로또 번호를 로또 구입 금액(input)만큼 발행
3. 계산
   - 당첨금액
   - 수익률 = 당첨금액 / 구입금액
4. Error
   - input 구입금액 : 1,000으로 나뉘지 읺는 수 입력 시
   - input 당첨번호 : 1~45 내의 숫자가 아닌 경우
   - input 당첨번호 : 6개 숫자를 입력하지 않은 경우
   - input 보너스 번호 : 1~45 내의 숫자가 아닌 경우
   - input 보너스 번호 : 1개 숫자를 입력하지 않은 경우
