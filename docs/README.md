# Lotto 기능 목록

## User class

- 사용자가 입력한 구매금액이 올바른 지 확인하는 기능
- 사용자가 로또를 n번 구매하는 기능
- 사용자의 순익을 계산하는 기능
- 사용자의 당첨 통계를 계산하는 기능

## Lotto class

- 생성된 숫자가 올바른 지 확인하는 기능
- 인자로 온 리스트와 비교하여 몇 등인지 판별하는 기능
- 


## LottoRanking (Enum)

- 등 수 판단 기능.


## LottoView class

- 구입 금액 입출력 기능
- 구입 금액이 올바른 지 테스트하는 기능
- 로또 출력 기능
- 당첨 통계 출력 기능
- 당첨 번호 입력받는 기능
