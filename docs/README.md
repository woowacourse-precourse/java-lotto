## 기능 목록

### User
- [x] 사용자에게 입력받는다.
  - [x] 로또 구입 금액을 입력받는다.
  - [x] 당첨 번호를 입력 받는다.
  - [x] 보너스 번호를 입력 받는다.

### Lotto
- [x] 로또를 발행한다.
- [x] 당첨 번호가 발행한 로또에서 몇 개나 있는지 확인한다.
- [x] 보너스 번호가 발행한 로또에서 몇 개나 있는지 확인한다.
- [x] 당첨 내역을 계산한다.
- [x] 수익률을 계산한다.

### Notice
- [x] 사용자에게 출력한다.
  - [x] 발행한 로또 수량을 출력한다.
  - [x] 발행한 로또 번호을 출력한다.
  - [x] 당첨 내역을 출력한다.
  - [x] 수익률을 출력한다.
