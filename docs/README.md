# ✅ 로또 게임

1~45 범위의 숫자를 임의로 선택해 맞추고 맞춘 숫자의 수에 따라 당첨금을 얻는 게임이다.

## ✏️ 과제 요구 사항

1. 클래스(객체)를 분리하는 연습
2. 도메인 로직에 대한 단위 테스트를 작성하는 연습

> 도메인 로직이란? 해결하고자하는 문제에 핵심이 되는 로직
>
> 해당 로직이 정말 문제를 해결하고 있는가?
>
> 입/출력, db 접근, 네트워킹은 도메인 로직이 아니라 서비스 로직이다.
