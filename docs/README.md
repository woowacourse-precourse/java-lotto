## 기능  목록
- [x] 1부터 45 사이에 있는 중복되지 않는 6개의 숫자 뽑기
- [x] 보너스 번호 1개 뽑기 (즉, 중복되지 않는 7개의 숫자 뽑기)
- [ ] 로또 구입 금액 입력받기
    - 1,000으로 나누어 떨어지지 않는 경우는 예외 처리 
- [ ] 당첨 번호 입력 받기 (쉼표 기준으로 구분)
    - 숫자는 6자리
    - 1부터 45 사이 숫자
    - 쉼표로 구분
    - 중복 없음
- [ ] 보너스 번호 입력 받기
    - 1부터 45 사이 숫자
    - 앞서 입력한 당첨 번호와 중복 없음
- [ ] 로또 구입 금액에 맞는 수량의 로또를 출력
    - 각 로또 안 숫자는 오름차순 정렬
- [ ] 당첨 금액 확인
  - [ ] 6자리 중에서 일치하는 개수 확인
  - [ ] 보너스 번호랑 일치하는 번호가 있는지 확인
- [ ] 당첨 내역 출력
    - 3개 이상 일치하는 경우만 출력
- [ ] 수익률 출력
    - 당첨 금액 / 구입 금액 (소수점 첫째 자리까지 출력)
