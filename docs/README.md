# 로또 게임 기능 list

### 로또(티켓)
- 6개의 수를 가지고 있다.

### 로또 생성자
- 로또 번호 랜덤 발행(o)
    - 로또 번호의 숫자 범위는 1~45까지이다.(o)
    - 한개의 로또에는 중복되지 않는 6개의 수이다.(o)
    - camp.nextstep.edu.missionutils Library를 이용한다.(o)


### 사용자
- 로또 구입 금액 입력하기 (o)
    - 금액이 1000원 단위여야 한다.
- 당첨 번호 입력하기(o)
    - 1~45 사이의 수여야 한다.(o)
    - 수는 총 6개로, 중복되지 않아야 한다.(o)
- 보너스 번호 입력하기(o)
    - 1~45 사이의 수여야 한다.(o)

### 로또 계산기
- 로또 번호와 당첨 번호 비교하기
    - 일치하는 번호 수 count(o)
- 보너스 번호와 로또 번호 비교하기(o)
- 수익률 계산하기(o)
    - 1등~5등 일치 개수 * 등수 별 수익금 / 원금 (%)
- 수익률과 일치번호 출력하기(o)
  - 등수는 enum으로 정의한다. 


### Input/Output 
- 입력
  - camp.nextstep.edu.missionutils Library를 이용한다.(o)
