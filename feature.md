# 기능 구현 목록

- 로또 구매 개수
    - 장당 1000원
    - 구입 금액 / 1000
    - [ERROR] 구입 금액이 1000으로 나누어 떨어지지 않는 경우 -> `IllegalArgumentException`


- 구매 로또 번호
    - `Randoms.pickUniqueNumbersInRange(1, 45, 6);`
    - 1 ≤ n ≤ 45
    - 6개
    - 중복되지 않음
    - 오름차순으로 정렬
    - 구매 개수만큼 반복


- 당첨 번호
    - 에러 발생 시, `IllegalArgumentException`
    - 당첨 번호
        - 쉼표 기준 구분
        - [ERROR] 입력 개수가 6개가 아닐 때
        - [ERROR] 숫자가 아닌 것 포함
        - [ERROR] 범위를 벗어난 숫자 포함
        - [ERROR] 중복 번호 포함
    - 보너스 번호
        - [ERROR] 입력 개수가 1개가 아닐 때
        - [ERROR] 숫자가 아닌 것
        - [ERROR] 범위를 벗어난 숫자
        - [ERROR] 당첨 번호와 중복인 숫자


- 번호 비교
    - 1등 : 6개 / 2,000,000,000원
    - 2등 : 5개 + 보너스 / 30,000,000원
    - 3등 : 5개 / 1,500,000원
    - 4등 : 4개 / 50,000원
    - 5등 : 3개 / 5,000원


- 당첨 내역 출력
    - 5등 -> 1등 순으로 출력


- 당첨 통계 (수익률)
    - 당첨 금액 / 구매 금액 * 100
    - 소수점 둘째자리에서 반올림 ex) `100.0%`, `51.5%`
