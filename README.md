# 미션 - 로또

## 🔍 기능 목록
- [x] 로또 구입 금액을 입력받는다
  - [x] [예외 상황] 숫자가 아닐 때
  - [x] [예외 상황] 1000으로 나누어 떨어지지 않을 때
  - [x] [예외 상황] 0을 입력 받았을 때
- [x] 로또를 발행하여 출력한다
- [X] 당첨 번호를 입력받는다
  - [X] [예외 상황] 쉼표를 기준으로 구분할 수 없을 때
  - [X] [예외 상황] 번호들이 숫자가 아닐 때
  - [X] [예외 상황] 번호의 범위가 1~45가 아닐 때
  - [X] [예외 상황] 번호가 6개가 아닐 때
  - [X] [예외 상황] 중복된 숫자가 있을 때
- [x] 보너스 번호를 입력받는다
  - [x] [예외 상황] 숫자가 아닐 때
  - [x] [예외 상황] 번호의 범위가 1~45가 아닐 때
  - [x] [예외 상황] 당첨 번호와 중복 될 때
- [x] 당첨 통계를 출력한다
- [x] 총 수익률을 출력한다
